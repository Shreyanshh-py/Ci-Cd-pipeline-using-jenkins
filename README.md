# Ci-Cd-pipeline-using-jenkins
I have deployed a 2-tier Flask application (Flask + MySQL) on AWS EC2 using Docker. I used Docker Compose to manage both containers with volumes and a custom network. The source code is hosted on GitHub, I've integrated Jenkins to Automate the Ci/Cd pipeline and the application automatically fetches the latest code from GitHub and deploys it on the EC2 instance once the Jenkins job is triggered.
